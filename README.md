## Описание программы

**Task Generator** — это GUI‑приложение для генерации случайных задач из списка. Программа позволяет добавлять задачи с указанием типа, управлять ими (удалять отдельные или очищать весь список), фильтровать по типам, а также сохранять и загружать данные из JSON‑файла.

### Возможности:
- ✅ Генерация случайной задачи из списка
- ✅ Добавление новых задач с указанием типа (учёба, спорт, работа, отдых, творчество, саморазвитие)
- ✅ Удаление отдельных задач и очистка всего списка
- ✅ Фильтрация задач по типу
- ✅ Сохранение всех задач в JSON‑файл
- ✅ Загрузка задач из JSON‑файла при запуске
- ✅ Просмотр полного списка всех задач с прокруткой
- ✅ Валидация вводимых данных (проверка на пустые поля, обязательный выбор типа задачи)
- ✅ Визуальная обратная связь через всплывающие сообщения и статусную строку

## Примеры использования

### Получение случайной задачи:
1. Запустите программу.
2. Нажмите кнопку **«Сгенерировать случайную задачу»**.
3. В верхней части окна появится случайная задача с указанием её типа.

### Добавление новой задачи:
1. Введите текст задачи в поле **«Текст задачи»** (например, «Выучить стихотворение»).
2. Выберите тип задачи из выпадающего списка: **«учёба»**, **«спорт»**, **«работа»**, **«отдых»**, **«творчество»** или **«саморазвитие»**.
3. Нажмите кнопку **«Добавить задачу»**.
4. Новая задача появится в списке внизу окна.
5. Поле ввода автоматически очистится — можно добавлять следующую задачу.

### Фильтрация задач:
1. В разделе **«Фильтры»** выберите нужный тип задачи из выпадающего списка **«По типу»**.
2. Нажмите **«Применить фильтр»** — список обновится, отображая только задачи выбранного типа.
3. Для сброса фильтров нажмите **«Сбросить»**.

### Удаление задач:
* Чтобы удалить одну задачу: выберите её в списке и нажмите **«Удалить выбранную»**.
* Чтобы очистить весь список: нажмите **«Очистить всё»** и подтвердите действие.


### Просмотр списка задач:
* Все добавленные задачи отображаются в прокручиваемом списке внизу окна в формате: **«Номер. Текст задачи (тип)»**.
* После добавления, удаления или фильтрации список автоматически обновляется.

## Валидация данных

Приложение проверяет:
* Поле **«Текст задачи»** не должно быть пустым. Если попытаться добавить пустую задачу, появится предупреждение.
* Обязательный выбор типа задачи из выпадающего списка. Если тип не выбран, появится сообщение с просьбой выбрать тип.
* При отсутствии задач в списке нажатие кнопки **«Сгенерировать случайную задачу»** вызовет предупреждение с просьбой сначала добавить задачи.
* Перед удалением всех задач приложение запрашивает подтверждение через диалоговое окно.
